    The extended practical retest

    As a disqualified driver or rider, a court can order that a candidate returns to being a learner driver and take an extended test. The test is twice as long as a standard test. The candidate must pass another theory test before booking the practical test

    When you book tests you should tell us if your candidate has any special needs or disabilities. This is so we can make reasonable adjustments for the test. More time might be allowed for the test if the candidate has certain special needs. This gives the examiner time to talk to the candidate about the condition and any adaptations fitted to the vehicle
